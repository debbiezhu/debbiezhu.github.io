# conclusion
- 超时:
	- 当请求超过时间限制还是没有被处理, 则请求直接被取消, 抛出超时异常.
	- 根据 tp99 都在 x 时间内被解决, 那么超时线可以设置为 99 线.
- 重试:
	- 为了避免再次生成请求消耗资源, 重试请求
	- 有次数限制
	- client 端带着 request id 进行请求
- 幂等:
	- 相同的 request id 多次请求, 最终结果应该一致.
