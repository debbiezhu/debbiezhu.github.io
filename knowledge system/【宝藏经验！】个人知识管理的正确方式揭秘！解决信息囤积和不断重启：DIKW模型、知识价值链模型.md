# index
- 信息囤积, 引发了焦虑与压力;
- dikw 模型:
	- data, 数据. 未经加工的原始材料, 没有任何意义; 
	- information, 信息. 能够为决策提供论据;
		- 把 data 图表化.
	- knowledge, 知识. 总结归纳而来;
	- wisdom, 智慧. 解决问题的能力.
- 为了筛选信息, 需要有方法:
	- 从身份出发, 以解决实际问题为导向, 来做知识管理
	- 把 问题 列出来, 看问题需要什么知识, 
	- 为了归纳这些知识, 需要收集什么信息.
